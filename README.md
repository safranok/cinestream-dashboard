# CineStream Content Analytics Dashboard

## Project Overview
CineStream Content Analytics is an interactive Streamlit dashboard for analyzing OTT content performance.

### Live Links
- Streamlit: https://cinestream-dashboard-xxhkvntf8hntbqwluqwwjc.streamlit.app/
- GitHub: https://github.com/safranok/cinestream-dashboard

## Features
- KPI Cards
- Interactive Filters
- Genre & Language Analysis
- ROI Analysis
- Quality Alerts
- CSV Export

## Technologies
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Plotly

## Run Locally
```bash
pip install -r requirements.txt
streamlit run cinestream_app.py
```
